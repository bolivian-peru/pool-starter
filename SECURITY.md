# Security Policy

## Supported versions

The latest minor release on `main` is supported. Patch releases are issued for security fixes only.

## Reporting a vulnerability

Please email **security@proxies.sx** with:

- A description of the issue
- Steps to reproduce
- The version / commit SHA affected
- Your suggested fix, if you have one

You'll get an acknowledgement within 48 hours. We'll work with you on a fix and agree on a coordinated disclosure timeline (typically 14–30 days depending on severity).

Please **do not** open a public GitHub issue for security problems.

## Scope

In scope:
- The `@proxies-sx/pool-sdk` package
- The `@proxies-sx/pool-portal-react` package
- The Next.js starter template in `apps/starter`
- Build and deploy scripts in this repo

Out of scope:
- Issues in upstream dependencies (report to those projects directly)
- The live Proxies.sx Pool Gateway itself — report at https://proxies.sx/security
